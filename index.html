Arquivo: é uma nomenclatura para definir uma informação ou software que foi armazenado no disco

Arquivos e pastas

Os arquivos e pastas do Windows 7 precisam de um nome para serem criados, esses nomes devem ter no máximo 260 caracteres, e o usuário pode utilizar qualquer caractere com exceção dos seguintes:

/ \ | < > * ? : "

Arquivos e pastas

Arquivos possuem extensões. Extensões são códigos normalmente de três caracteres e estão associadas com o formato do arquivo. São separadas do nome do arquivo pelo caractere ponto "." veja abaixo:

.exe, .doc, .docx, .xls, .xlsx, .odt, .pdf, .html, .java, .cs, .gif, .jpg

Sistemas de arquivos

São as regras e os padrões do Windows 7 para gerenciar seus arquivos e pastas

O Windows 7 utiliza como tipo de sistema de arquivos o NTFS (Network File System)

Outros sistemas de arquivos conhecidos são: FAT(File Alocation Table), EXT(Extended File System) esse último muito utilizado pelo  Linux.

Backup e Restauração

Por padrão a imagem do sistema inclui toda  a unidade de disco necessária a execução do Windows. Mas o usuário pode selecionar somente os arquivos que o interessam na hora de fazer o bakcup

Painel de Controle - Sistema

Backup e Restauração

Tipos de Backup

  • Backup de cópia - Copia os arquivos selecionados sem alterar seus atributos de arquivos.
  • Backup diário - Copia somente arquivos que foram alterados no dia da execução do backup, sem alterar seu atributo de arquivo
  • Backup diferencial - Copia arquivos criados ou alterados desde o último backup normal ou incremental. Não marca os arquivos que passaram por backup.
  • Backup incremental - Copia arquivos criados ou alterados desde o último backup normal ou incremental. Marca os arquivos como arquivos que passaram pelo backup
  • Backup normal - Copia os arquivos selecionados e os marca como arquivos que passaram pelo bakcup
